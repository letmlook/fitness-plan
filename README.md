## 🎯 本月核心目标

1. **体重**：92kg → 88kg（-4kg）
2. **饮食**：戒奶茶/甜饮料，三餐定时
3. **运动**：每周 5 天有氧 + 2 天力量
4. **习惯**：每天喝水 2000ml，11点前睡觉

---

## 🍽️ 饮食计划

### 每日摄入目标：约 1700-1800 kcal

```
早餐（300-400kcal）  7:00 - 8:00
午餐（600-700kcal） 12:00 - 13:00
晚餐（400-500kcal） 18:00 - 19:00
加餐（100-200kcal） 15:00 / 21:00
```

### 食物选择

| 餐次 | 推荐食物 |
|------|----------|
| 早餐 | 燕麦30g + 鸡蛋2个 + 牛奶200ml / 红薯200g + 鸡蛋 |
| 午餐 | 糙米150g + 鸡胸肉150g + 蔬菜随意 / 牛肉150g + 米饭 |
| 晚餐 | 南瓜/红薯200g + 鱼肉200g + 蔬菜 / 豆腐150g + 绿叶菜 |
| 加餐 | 苹果1个 / 香蕉1根 / 坚果20g / 无糖酸奶100g |

### ❌ 避免

- 奶茶、碳酸饮料、果汁
- 油炸食品、烧烤
- 白米饭、面条（晚）限量
- 零食、薯片、巧克力

## 🏃 运动计划

### 周一/三/五：有氧燃脂

| 时间 | 内容 | 消耗 |
|------|------|------|
| 早上 | 快走/慢跑 30-40 分钟 | 200-300 kcal |
| 晚上 | 帕梅拉/跳绳 20 分钟 | 150-200 kcal |

### 周二/四：力量训练

| 部位 | 动作 | 组数 |
|------|------|------|
| 胸 | 俯卧撑 | 3×15 |
| 背 | 跪姿划船（矿泉水瓶） | 3×15 |
| 腿 | 深蹲 | 3×20 |
| 腹 | 平板支撑 | 3×45秒 |
| 腹 | 卷腹 | 3×20 |

### 周六：混合训练

- HIIT 20 分钟（开合跳+高抬腿+波比跳）
- 拉伸 10 分钟

### 周日：休息 / 轻度活动

- 散步 30 分钟 or 拉伸 15 分钟

## 💡 小贴士

1. **喝水**：饭前喝一杯水，减小食欲
2. **细嚼慢咽**：每口咀嚼 20 次
3. **早睡**：熬夜会导致皮质醇升高，不利于减脂
4. **记录**：每天记录体重和饮食，效果翻倍
5. **不要饿着**：饿太久会掉肌肉，反弹更快
